Image Optimization

Responsive Images

Just add data-optimized to any <img> tag. AbsoluteJS rewrites it at build time with the optimization endpoint URL, responsive srcset, lazy loading, and async decoding — automatically.

Landscape
1200×800 with responsive sizes. The build rewrites src and adds srcset.
src: /assets/jpg/landscape.jpg
Portrait
600×900 portrait. Width/height preserved for CLS prevention.
src: /assets/jpg/portrait.jpg
Square
800×800 square. Same data-optimized attribute, automatic optimization.
src: /assets/jpg/square.jpg

Fill Mode

For fill mode in HTML, use CSS on the parent and data-optimized on the img.

Unoptimized (no data-optimized)

Without the data-optimized attribute, the image is served as-is. Just a normal <img> tag — no build-time rewriting.

Unoptimized
Original JPG served directly. No data-optimized = no transformation.
unoptimized: true
